# Help

This repository contains the source codes for old machine projects for Programming With Structured Data Types and Discrete Structures (Term 2, A.Y. 2023-24).

이 저장소에는 구조화된 데이터 유형 및 이산 구조 프로그래밍을 위한 오래된 기계 프로젝트의 소스 코드가 포함되어 있습니다(2기, A.Y. 2023-24).

## Language Specifications

The source codes for both CCPROG2 and CCDSTRU were written in C following the [**C99** standard](https://en.cppreference.com/w/c/99).

CCPROG2와 CCDSTRU의 소스 코드는 [**C99** 표준을](https://en.cppreference.com/w/c/99) 따라 C로 작성되었습니다.

The CCDSTRU machine project specifications allow you to write it in Java; a [Java **SE 8**](https://docs.oracle.com/javase/8/docs/api/) source code is also available. However, the source code is made to resemble a game, as opposed to matching the specifications.

CCDSTRU 머신 프로젝트 사양을 사용하면 Java로 작성할 수 있으며, [**Java SE 8**](https://docs.oracle.com/javase/8/docs/api/) 소스 코드도 사용할 수 있습니다. 그러나 소스 코드는 사양을 맞추는 대신 게임과 비슷하게 만들어졌습니다.

### Summary

- C Standard: C99
- Java Release: SE 8
