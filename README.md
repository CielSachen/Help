# Help

This repository contains the source code for an old machine project for Discrete Structures (Term 2, A.Y. 2023-24).

## Language Specifications

The source codes were written in two languages: C and Java. The C source code is bare bones and as close to the specifications as possible, while the Java source code is detailed and as game-like as possible.

### Language Standards or Releases

1. **C** - ISO/IEC 9899:1999 (C99)
2. **Java** - Java SE 8
